# 编码与绘图规范

**语言：** [English](AGENTS.en.md) | [中文](AGENTS.zh-CN.md)

优先在 conda 环境中使用软件和包，不直接运行系统版本的 R 或 Python。  
绘图优先使用 R 语言。  
绘图时不添加背景网格线，不添加上方黑线框和右方黑线框。

配色模板：

```python
color_palette = {
    1: "#00BFAE",
    2: "#1F77B4",
    3: "#9467BD",
    4: "#FF7F0E",
    5: "#D62728",
    6: "#F08080",
    7: "#8B4513",
    8: "#228B22",
    9: "#90EE90",
    10: "#00008B",
    11: "#DDA0DD",
    12: "#006400",
    13: "#8B0000",
    14: "#ADD8E6",
}
```

构建分析脚本时优先考虑多核并行，减少运行时间；最大程度利用服务器资源；后台运行并定时输出任务进度。

隐私：公开文档与提交中只使用样本编号（T/C/O），不写真实物种中文名或拉丁名。
